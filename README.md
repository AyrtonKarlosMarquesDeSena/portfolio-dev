Portfólio profissional desenvolvido com **React**, **CSS Modules** e **React Router**. Apresenta meus projetos, habilidades e experiências como desenvolvedor FullStack.

---

## 📌 Sobre

Este portfólio foi criado para apresentar minha trajetória como desenvolvedor FullStack, com foco em:

- **Design responsivo** (mobile-first)
- **Navegação fluida** entre páginas
- **Cards interativos** com meus projetos
- **Menu mobile** com animação suave
- **Seções**: Home, Sobre, Projetos e Contato

---

## 🛠️ Tecnologias

| Tecnologia | Descrição |
|------------|-----------|
| **React 19** | Biblioteca para construção da interface |
| **React Router DOM** | Navegação entre páginas |
| **CSS Modules** | Estilização encapsulada por componente |
| **React Icons** | Ícones personalizados |
| **Vite** | Build rápido e desenvolvimento |

---

## 🚀 Como Executar

```bash
# 1. Clone o repositório
git clone https://github.com/AyrtonKarlosMarquesDeSena/portfolio-dev.git

# 2. Acesse a pasta
cd portfolio-dev

# 3. Instale as dependências
npm install

# 4. Rode o projeto
npm run dev
